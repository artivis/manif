The base class of the SE_2_3 group.

Note

See Appendix A2 in the paper "The Invariant Extended Kalman filter as a stable observer". However, note that the serialization used in that paper is different from that defined below The paper uses a SE_2_3 definition as, X = |R v p| | 1 | | 1| with a vector space serialization as (w, a, v) Instead, here we define the SE_2_3 to be, X = |R p v| | 1 | | 1| with a vector space serialization as (v, w, a)
